## The Suzuki-Kasami algorithm is a token-based algorithm for achieving mutual exclusion in distributed systems.

So, the node which has the privilege it can use the critical section and if it does not have one it cannot. If a process wants to enter its critical section and it does not have the token, it broadcasts a request message to all other processes in the system. The process that has the token, if it is not currently in a critical section, will then send the token to the requesting process.
